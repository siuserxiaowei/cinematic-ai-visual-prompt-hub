# 电影级视觉提示词学习站

一个面向 AI 图片、AI 视频和分镜创作的公开 GitHub Pages 学习站。项目会把影视基础知识、镜头语言、运镜、布光、构图、相机与影调、分镜脚本和 AI 生成提示词整理成可搜索、可筛选、可复制、可溯源的学习资料库。

## Scope

- 公开站点：GitHub Pages 发布。
- 内容来源：官方文档、影视教学、GitHub 仓库、中文社区、视频平台和公开社交内容。
- 内容处理：只沉淀原创化摘要、术语卡片、提示词模板和来源索引，不搬运受版权保护全文或第三方素材。
- 设计方向：参考 RunningHub 电影级视觉资产提示词库的黑色电影感、全屏视觉开场、横向分类卡、沉浸式详情弹层和复制提示词体验。
