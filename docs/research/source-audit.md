# Source Audit

Checked on 2026-07-01.

## Coverage

- Official/vendor docs: Runway, Google DeepMind/Veo, Google Cloud, OpenAI Developers, Alibaba Cloud/Wan, 阿里云, Kling AI, MiniMax, Adobe Firefly, Luma Labs.
- Film education: StudioBinder, Adobe, Boords, UCI film glossary, ARRI, Kodak, Blackmagic Design.
- GitHub: Wan2.1, HunyuanVideo prompt handbook, DirectorsConsole, awesome-ai-video-prompts.
- Chinese web: Zhihu, WaytoAGI, 少数派, 人人都是产品经理, 博客园, CSDN, 数英网, 掘金.
- Video/social: B站, 抖音 web results, YouTube, X/Twitter public snippets.

## Caveats

- X/Twitter was partially restricted from public browsing; X items are marked as snippet/trend references and should be rechecked in a signed-in browser before exact claims.
- 抖音 and 小红书 public search are dynamic; 抖音 entries are trend signals only. No stable 小红书 source was added.
- YouTube/B站 items are learning references, not authoritative definitions.
- Sora-related material is retained as prompt-design reference because product/API status can drift.
- Third-party prompt banks are summarized as patterns; no large prompt lists are copied.
- SEO-heavy, affiliate-heavy, watermark-removal, uncensored, reposted PDF, and unverifiable prompt dump sources were excluded.

## Source Policy

The site stores original summaries, learning values, term mappings, and prompt patterns. It does not republish third-party article bodies, screenshots, video transcripts, or full prompt banks.
